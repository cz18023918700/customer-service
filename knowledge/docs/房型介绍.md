# 静享时空 房型介绍

## 翰林府店

我们有三种类型的包厢，满足不同需求：

### 🎤 欢唱大包厢（666号、888号房）
- 可唱歌，配备专业KTV点歌系统
- 适合6-10人朋友聚会、生日派对、团建
- 内有沙发、茶几、灯光氛围
- 可到店内无人超市自取平价酒水零食

### 🎤 尊享大包厢（999号房）
- 最大最豪华的包厢，可唱歌
- 适合8-15人大型聚会
- 配置最高，音响效果最好
- 热门房型，建议提前预约

### 🛋️ 榻榻米中包厢（333号、555号房）
- 日式榻榻米风格，舒适放松
- 适合3-5人小聚、聊天、桌游
- 私密性好，环境安静

### 🔒 私密小包厢（111号、222号房）
- 小而温馨的私密空间
- 适合2-3人、情侣约会
- 安静不被打扰

### 💺 雅座
- 开放式休闲座位
- 适合1-3人休闲、看书、小坐
- 价格实惠

## 恒大城11栋店

### 🍵 茶室
- 两间独立茶室
- 适合喝茶聊天、商务洽谈、安静聚会
- 提供基础茶具
