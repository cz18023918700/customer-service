# 静享时空 常见问题

## 关于预约

### 需要提前预约吗？
不一定。如果有空房可以直接下单使用。但晚上时段的大包厢比较抢手，建议提前在小程序预约。

### 可以当天预约吗？
可以，小程序上实时显示空房情况，随时可以预约。

### 怎么取消预约？
在小程序"我的订单"里取消。提前2小时以上取消免费，2小时内取消可能收取一定费用。

## 关于费用

### 怎么付款？
通过微信小程序在线支付，支持微信支付。

### 有最低消费吗？
没有最低消费，按房型和时长正常计费。

### 超时怎么收费？
中小包厢和茶室超时按每小时计费（8元或5元），大包厢如果超出预定时段会按下一时段价格计算。建议提前在小程序续费。

## 关于设备

### 话筒没有声音怎么办？
1. 检查话筒底部开关是否打开
2. 长按话筒电源键3秒重启
3. 检查话筒电池是否有电（备用电池在抽屉里）
4. 还是不行请联系客服，我们会尽快处理

### 空调不制冷/制热怎么办？
1. 检查遥控器是否有电
2. 确认模式是否正确（制冷/制热）
3. 等待3-5分钟让空调启动
4. 还是不行请联系客服

### 投影/电视不亮怎么办？
1. 检查电源是否接通
2. 按遥控器电源键
3. 尝试切换信号源
4. 还是不行请联系客服

## 关于安全

### 店里有监控吗？
公共区域有监控，包厢内没有，充分保护你的隐私。

### 东西丢了怎么办？
联系客服说明情况，我们会帮你查看和寻找。贵重物品请随身携带。

### 遇到紧急情况怎么办？
店内有紧急联系电话和安全出口标识，如遇紧急情况请拨打110/119/120。

## 其他

### 可以带宠物吗？
抱歉，为了保持环境清洁和其他顾客的体验，暂不允许携带宠物。

### 有停车位吗？
恒大城小区内有停车位，具体以小区物业安排为准。

### 适合带小孩吗？
可以的，但请家长全程看护，未成年人不可单独使用包厢。
