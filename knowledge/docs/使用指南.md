# 静享时空 使用指南

## 第一次来？看这里就够了

### 步骤一：选房下单
1. 打开微信，搜索"静享时空"小程序
2. 选择想要的包厢类型和时间
3. 微信支付完成下单

### 步骤二：到店开门
1. 按照导航到达店铺
2. 打开小程序，点击"开门"
3. 门会自动解锁，进入包厢

### 步骤三：开始使用
- 大包厢：打开电视和点歌系统，就可以唱歌了
- 中小包厢/茶室：直接享受空间
- 需要酒水零食：到无人超市自助扫码购买

### 步骤四：结束离开
1. 使用完毕后整理好个人物品
2. 离开房间，门会自动锁上
3. 超时会自动按超出时间计费

## 常见操作问题

### 门打不开怎么办？
1. 确认订单已支付成功
2. 检查手机蓝牙和网络是否打开
3. 在小程序里重新点击"开门"
4. 如果还是不行，联系客服

### 点歌系统怎么用？
1. 打开包厢内的电视
2. 使用遥控器或扫码连接点歌
3. 搜索歌曲，点击即可播放

### 空调/灯光怎么调？
- 空调遥控器在墙上的收纳盒里
- 灯光开关在门边

### 可以自带食物吗？
- 可以自带食物和非酒精饮料
- 酒水建议在店内无人超市购买，价格很平价
- 请保持包厢清洁

### 可以续时吗？
- 可以，在小程序上直接续费即可
- 如果后面没有人预约，可以一直续
